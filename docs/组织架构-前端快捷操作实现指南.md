# 组织架构 - 前端快捷操作实现指南

## 概述

本文档说明如何在前端组织架构卡片上添加快捷操作按钮，让用户可以便捷地：
- 设置 Agent 为负责人（主/副）
- 标记岗位为管理岗位
- 设置部门的管理部门
- 查看推荐的经理

### 新增 API 方法

```typescript
// web/src/lib/api.ts

class API {
  // ... 现有方法 ...

  // ==================== Quick Actions ====================

  /**
   * 获取推荐的经理
   */
  async getRecommendedManager(agentId: number): Promise<Agent | null> {
    try {
      const response = await this.fetch(`/api/agents/${agentId}/recommended-manager`);
      return response;
    } catch (error) {
      // 404 表示没有推荐的经理
      if ((error as any)?.status === 404) {
        return null;
      }
      throw error;
    }
  }

  /**
   * 设置 Agent 为负责人
   */
  async setAgentAsLeader(
    agentId: number,
    leadershipRole: 'primary' | 'deputy' | 'none'
  ): Promise<Agent> {
    return this.fetch(`/api/agents/${agentId}/set-leader`, {
      method: 'POST',
      body: JSON.stringify({ leadership_role: leadershipRole }),
    });
  }

  /**
   * 设置岗位为管理岗位
   */
  async setPositionAsManagement(
    positionId: number,
    isManagement: boolean
  ): Promise<Position> {
    return this.fetch(`/api/positions/${positionId}/set-management`, {
      method: 'POST',
      body: JSON.stringify({ is_management: isManagement }),
    });
  }

  /**
   * 设置部门的管理部门
   */
  async setManagingDepartment(
    departmentId: number,
    managingDepartmentId: number | null
  ): Promise<Department> {
    return this.fetch(`/api/departments/${departmentId}/set-managing-department`, {
      method: 'POST',
      body: JSON.stringify({ managing_department_id: managingDepartmentId }),
    });
  }
}

export const api = new API();
```

### 组件示例

```tsx
// web/src/pages/OrganizationPage/AgentCard.tsx

import { Agent } from '@/lib/api';
import { api } from '@/lib/api';
import { Button, Dropdown, Menu, Badge, Tooltip } from 'antd';
import { CrownOutlined, UserOutlined, MoreOutlined } from '@ant-design/icons';
import { useState } from 'react';
import { useTranslation } from 'react-i18next';

interface AgentCardProps {
  agent: Agent;
  onUpdate: () => void;
}

export function AgentCard({ agent, onUpdate }: AgentCardProps) {
  const { t } = useTranslation();
  const [loading, setLoading] = useState(false);

  const handleSetLeader = async (role: 'primary' | 'deputy' | 'none') => {
    setLoading(true);
    try {
      await api.setAgentAsLeader(agent.id, role);
      onUpdate(); // 刷新组织架构树
    } catch (error) {
      console.error('Failed to set leader:', error);
      // 显示错误提示
    } finally {
      setLoading(false);
    }
  };

  const leaderMenu = (
    <Menu
      items={[
        {
          key: 'primary',
          label: t('org.setPrimaryLeader'),
          icon: <CrownOutlined />,
          disabled: agent.leadership_role === 'primary',
          onClick: () => handleSetLeader('primary'),
        },
        {
          key: 'deputy',
          label: t('org.setDeputyLeader'),
          icon: <UserOutlined />,
          disabled: agent.leadership_role === 'deputy',
          onClick: () => handleSetLeader('deputy'),
        },
        {
          key: 'none',
          label: t('org.removeLeader'),
          disabled: agent.leadership_role === 'none',
          onClick: () => handleSetLeader('none'),
        },
      ]}
    />
  );

  return (
    <div className="agent-card">
      <div className="agent-header">
        <div className="agent-info">
          {/* 显示领导角色标识 */}
          {agent.leadership_role === 'primary' && (
            <Badge.Ribbon text={t('org.primaryLeader')} color="gold">
              <span>{agent.display_name || agent.name}</span>
            </Badge.Ribbon>
          )}
          {agent.leadership_role === 'deputy' && (
            <Badge.Ribbon text={t('org.deputyLeader')} color="blue">
              <span>{agent.display_name || agent.name}</span>
            </Badge.Ribbon>
          )}
          {agent.leadership_role === 'none' && (
            <span>{agent.display_name || agent.name}</span>
          )}
        </div>

        {/* 快捷操作菜单 */}
        <Dropdown menu={{ items: leaderMenu.props.items }} trigger={['click']}>
          <Button
            icon={<MoreOutlined />}
            loading={loading}
            size="small"
            type="text"
          />
        </Dropdown>
      </div>

      {/* Agent 其他信息 */}
      <div className="agent-body">
        {/* ... */}
      </div>
    </div>
  );
}
```

## 3. Position 卡片快捷操作

```tsx
// web/src/pages/OrganizationPage/PositionCard.tsx

import { Position } from '@/lib/api';
import { api } from '@/lib/api';
import { Switch, Tooltip } from 'antd';
import { CrownOutlined } from '@ant-design/icons';
import { useState } from 'react';
import { useTranslation } from 'react-i18next';

interface PositionCardProps {
  position: Position;
  onUpdate: () => void;
}

export function PositionCard({ position, onUpdate }: PositionCardProps) {
  const { t } = useTranslation();
  const [loading, setLoading] = useState(false);

  const handleToggleManagement = async (checked: boolean) => {
    setLoading(true);
    try {
      await api.setPositionAsManagement(position.id, checked);
      onUpdate();
    } catch (error) {
      console.error('Failed to toggle management position:', error);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="position-card">
      <div className="position-header">
        <div className="position-name">
          {position.is_management_position && (
            <CrownOutlined style={{ color: '#faad14', marginRight: 8 }} />
          )}
          {position.name}
        </div>

        {/* 管理岗位开关 */}
        <Tooltip title={t('org.setManagementPosition')}>
          <Switch
            checked={position.is_management_position}
            onChange={handleToggleManagement}
            loading={loading}
            size="small"
            checkedChildren={t('org.managementPosition')}
          />
        </Tooltip>
      </div>

      {/* Position 其他信息 */}
      <div className="position-body">
        {/* 显示该岗位下的 Agents */}
        {position.agents?.map(agent => (
          <AgentCard key={agent.id} agent={agent} onUpdate={onUpdate} />
        ))}
      </div>
    </div>
  );
}
```

## 4. Department 卡片快捷操作

```tsx
// web/src/pages/OrganizationPage/DepartmentCard.tsx

import { Department } from '@/lib/api';
import { api } from '@/lib/api';
import { Select, Button } from 'antd';
import { LinkOutlined } from '@ant-design/icons';
import { useState } from 'react';
import { useTranslation } from 'react-i18next';

interface DepartmentCardProps {
  department: Department;
  allDepartments: Department[]; // 所有部门列表（用于选择管理部门）
  onUpdate: () => void;
}

export function DepartmentCard({ 
  department, 
  allDepartments, 
  onUpdate 
}: DepartmentCardProps) {
  const { t } = useTranslation();
  const [loading, setLoading] = useState(false);
  const [showManagingSelect, setShowManagingSelect] = useState(false);

  const handleSetManagingDepartment = async (managingDeptId: number | null) => {
    setLoading(true);
    try {
      await api.setManagingDepartment(department.id, managingDeptId);
      setShowManagingSelect(false);
      onUpdate();
    } catch (error) {
      console.error('Failed to set managing department:', error);
    } finally {
      setLoading(false);
    }
  };

  // 过滤可选的管理部门（排除自己和子部门）
  const availableDepartments = allDepartments.filter(
    d => d.id !== department.id && d.parent_id !== department.id
  );

  return (
    <div className="department-card">
      <div className="department-header">
        <div className="department-name">{department.name}</div>

        {/* 管理部门设置 */}
        {showManagingSelect ? (
          <Select
            style={{ width: 200 }}
            placeholder={t('org.setManagingDepartment')}
            value={department.managing_department_id}
            onChange={handleSetManagingDepartment}
            loading={loading}
            allowClear
            options={[
              { label: t('common.none'), value: null },
              ...availableDepartments.map(d => ({
                label: d.name,
                value: d.id,
              })),
            ]}
          />
        ) : (
          <Button
            icon={<LinkOutlined />}
            onClick={() => setShowManagingSelect(true)}
            size="small"
            type="text"
          >
            {department.managing_department_id
              ? t('org.changeManagingDepartment')
              : t('org.setManagingDepartment')}
          </Button>
        )}
      </div>

      {/* 显示当前管理部门 */}
      {department.managing_department_id && (
        <div className="department-managing-info">
          {t('org.managedBy')}:{' '}
          {allDepartments.find(d => d.id === department.managing_department_id)?.name}
        </div>
      )}

      {/* Department 其他信息 */}
      <div className="department-body">
        {/* 显示该部门下的 Positions */}
        {department.positions?.map(position => (
          <PositionCard key={position.id} position={position} onUpdate={onUpdate} />
        ))}
      </div>
    </div>
  );
}
```

```tsx
// web/src/pages/OrganizationPage/OrgTree.tsx

import { Agent, Position, Department } from '@/lib/api';
import { Tree } from 'antd';
import { CrownOutlined, TeamOutlined, LinkOutlined } from '@ant-design/icons';
import { useTranslation } from 'react-i18next';

interface OrgTreeProps {
  departments: Department[];
  onSelectNode: (node: any) => void;
}

export function OrgTree({ departments, onSelectNode }: OrgTreeProps) {
  const { t } = useTranslation();

  const buildTreeData = (dept: Department): any => {
    return {
      key: `dept-${dept.id}`,
      title: (
        <span>
          {dept.name}
          {/* 显示管理关系 */}
          {dept.managing_department_id && (
            <LinkOutlined style={{ marginLeft: 8, color: '#1890ff' }} />
          )}
        </span>
      ),
      children: [
        // Positions
        ...(dept.positions || []).map((pos: Position) => ({
          key: `pos-${pos.id}`,
          title: (
            <span>
              {pos.is_management_position && (
                <CrownOutlined style={{ color: '#faad14', marginRight: 4 }} />
              )}
              {pos.name}
            </span>
          ),
          children: (pos.agents || []).map((agent: Agent) => ({
            key: `agent-${agent.id}`,
            title: (
              <span>
                {agent.display_name || agent.name}
                {agent.leadership_role === 'primary' && ' 👑'}
                {agent.leadership_role === 'deputy' && ' 🎖️'}
              </span>
            ),
            isLeaf: true,
          })),
        })),
      ],
    };
  };

  return (
    <Tree
      treeData={departments.map(buildTreeData)}
      onSelect={(keys, info) => onSelectNode(info.node)}
    />
  );
}
```

## 8. 测试场景

### 场景 1：设置主负责人

1. 用户点击 Agent 卡片上的「更多」按钮
2. 选择「设为主负责人」
3. 系统检查同岗位是否已有主负责人
4. 如果有冲突，显示错误提示
5. 成功后刷新组织架构树，显示金色皇冠标识

### 场景 2：标记管理岗位

1. 用户点击 Position 卡片上的「管理岗位」开关
2. 系统更新数据库
3. 卡片显示皇冠图标
4. 该岗位下的 Agents 自动成为其他岗位的候选经理

### 场景 3：设置管理部门

1. 用户点击 Department 卡片上的「设置管理部门」
2. 下拉选择另一个部门
3. 系统验证不会造成循环引用
4. 显示管理关系链接图标

### 场景 4：创建 Agent 时推荐经理

1. 用户选择岗位创建 Agent
2. 系统自动查找该岗位的负责人 Agent
3. 如果找到，自动填充到「直属负责人」字段
4. 显示「推荐」标签
5. 用户可以手动修改

## 9. 后续优化建议

## 10. API 错误处理

```typescript
// web/src/lib/api.ts

// 统一错误处理
try {
  await api.setAgentAsLeader(agentId, 'primary');
} catch (error) {
  if (error.status === 409) {
    // 冲突：同岗位已有主负责人
    message.error(t('org.primaryLeaderConflict'));
  } else if (error.status === 404) {
    // Agent 不存在
    message.error(t('org.agentNotFound'));
  } else {
    // 其他错误
    message.error(t('common.operationFailed'));
  }
}
```

---

**实现优先级**：
1. 高优先级：Agent 设置负责人、Position 标记管理岗位（核心功能）
2. 中优先级：Department 设置管理部门、创建时推荐经理（辅助功能）
3. 低优先级：可视化关系图、批量操作（锦上添花）

## 附录: 快速问题排查

### 如果 Analytics 仍显示 "null"

1. 检查浏览器开发者工具 Console
2. 查看 Network 标签，检查 `/api/analytics/usage` 响应
3. 重新编译前端: `npm run build:web`
4. 清除缓存重启应用

### 如果历史会话为空

1. 检查数据库:
   ```bash
   sqlite3 ~/Library/Application\ Support/hermes-agent-electron/state.db \
     "SELECT COUNT(*) FROM sessions;"
   ```
2. 检查应用使用的路径（查看启动日志）

### 如果应用无法启动

1. 检查 Python 是否安装
2. 查看完整错误日志
3. 运行: `npm run build:all` 重新编译

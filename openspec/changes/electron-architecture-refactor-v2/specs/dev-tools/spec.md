## 新增需求

### 需求: 实时日志流
开发工具必须提供从 Gateway 进程到 Renderer 的实时日志流。

#### 场景: 流式传输 Gateway 日志
- **当** 调用 dev:streamLogs IPC 处理器时
- **则** 系统必须 tail gateway.log 并通过 'dev:log' IPC 事件发送更新

#### 场景: 清理时停止流
- **当** 日志查看器组件卸载时
- **则** 系统必须停止 tail 进程并清理资源

#### 场景: 日志过滤
- **当** 用户在日志查看器中输入搜索查询时
- **则** 只有匹配的日志行必须被显示

#### 场景: 日志导出
- **当** 用户点击"导出日志"按钮时
- **则** 系统必须下载完整的 gateway.log 文件

### 需求: 服务仪表板
开发工具必须显示所有注册服务的实时状态和指标。

#### 场景: 列出所有服务
- **当** 开发工具页面加载时
- **则** 仪表板必须显示所有服务及其 id、状态和健康状态

#### 场景: 显示服务指标
- **当** 用户点击服务时
- **则** 仪表板必须显示来自 service.getMetrics() 的指标（如果已实现）

#### 场景: 手动重启
- **当** 用户点击服务的"重启"按钮时
- **则** 系统必须调用 service.stop() 后再调用 service.start()

#### 场景: 依赖图可视化
- **当** 用户查看服务仪表板时
- **则** UI 必须显示服务依赖关系的可视化图

### 需求: IPC 检查器
开发工具必须提供 IPC 处理器的检查和测试功能。

#### 场景: 列出已注册的处理器
- **当** IPC 检查器加载时
- **则** 系统必须显示所有通过 IpcRegistry 注册的处理器

#### 场景: 查看处理器 schema
- **当** 用户点击处理器时
- **则** 系统必须显示输入/输出 schema 和限流配置

#### 场景: 手动测试处理器
- **当** 用户输入测试输入并点击"调用"时
- **则** 系统必须调用 IPC 处理器并显示响应

#### 场景: 查看调用历史
- **当** 开发期间进行 IPC 调用时
- **则** 检查器必须记录每次调用的时间戳、输入、输出和持续时间

### 需求: 性能指标查看器
开发工具必须显示启动时间、健康检查延迟和断路器状态。

#### 场景: 显示启动指标
- **当** 指标页面加载时
- **则** 系统必须显示 Gateway 启动时间和尝试次数

#### 场景: 显示健康检查指标
- **当** 指标页面加载时
- **则** 系统必须显示健康检查的平均延迟、P95、P99

#### 场景: 显示断路器状态
- **当** 指标页面加载时
- **则** 系统必须显示当前断路状态（CLOSED/OPEN/HALF_OPEN）和失败计数

#### 场景: 实时更新
- **当** 指标变化时
- **则** UI 必须自动更新而无需手动刷新

### 需求: VSCode 调试配置
项目必须包含用于调试 Main、Renderer 和 Python 进程的 launch.json 配置。

#### 场景: 调试 Main 进程
- **当** 使用"Debug Main Process"启动配置时
- **则** VSCode 必须使用 --inspect 标志附加调试器到 Electron main 进程

#### 场景: 调试 Renderer 进程
- **当** 使用"Debug Renderer Process"启动配置时
- **则** VSCode 必须附加 Chrome 调试器到 http://localhost:5173

#### 场景: 调试 Python Gateway
- **当** 使用"Attach to Python Gateway"启动配置时
- **则** VSCode 必须附加 Python 调试器到端口 5678（debugpy）

#### 场景: 复合调试
- **当** 使用"Debug Both (Main + Renderer)"启动配置时
- **则** VSCode 必须同时启动两个调试器

### 需求: 键盘快捷键访问
开发工具页面必须通过 Cmd+Shift+D（或 Windows/Linux 上的 Ctrl+Shift+D）访问。

#### 场景: 切换开发工具
- **当** 用户按下 Cmd+Shift+D 时
- **则** 窗口必须导航到 /dev-tools 路由

#### 场景: 全局快捷键注册
- **当** 应用启动时
- **则** 系统必须为开发工具访问注册 globalShortcut

#### 场景: 防止冲突
- **当** 注册开发工具快捷键时
- **则** 系统不得与现有的 Electron DevTools 快捷键（Cmd+Option+I）冲突

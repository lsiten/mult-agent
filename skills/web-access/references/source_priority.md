# Source Priority

## Preferred Sources

优先顺序：

1. 官方站点与官方文档
2. 原始公告、白皮书、标准、论文、政府页面
3. 主流权威媒体或知名技术文档站
4. 聚合与转述内容

## Verification Notes

- 查询“最新”信息时，至少核对发布日期或更新时间
- 同时存在“报道日期”和“事件日期”时，回答里优先写事件日期，并补充报道日期
- 推荐、价格、政策、版本号这类问题，尽量不要只看一个页面
- 技术问题优先引用官方文档，不足时再补社区资料

## Save Suggestions

当用户要求保存时：

- 摘要型内容保存为 `.md`
- 结构化字段保存为 `.json`
- 原始链接较多时，先做链接清单再补摘要
